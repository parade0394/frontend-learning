弹性尺寸 - minmax()

使用 minmax() 函数设置弹性尺寸范围

.grid {
  grid-template-columns:
    minmax(100px, 200px)
    minmax(150px, 1fr)
    minmax(100px, 300px);
}
列 1 100-200px
列 2 150px-1fr
列 3 100-300px
