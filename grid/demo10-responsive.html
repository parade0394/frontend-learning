Demo 10: 响应式布局

知识点：结合 auto-fit、minmax() 和媒体查询创建响应式网格。

桌面：每列最小200px，自动适应

平板（≤768px）：每列最小150px

手机（≤480px）：单列布局

试着调整浏览器窗口大小，观察布局变化！

.grid {
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

@media (max-width: 768px) {
  .grid { grid-template-columns: repeat(auto-fit, minmax(150px, 1fr)); }
}
