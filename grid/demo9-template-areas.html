Demo 9: 模板区域 - grid-template-areas

知识点：使用 grid-template-areas 创建命名的网格区域。

这是创建复杂布局最直观的方式，代码可读性极高！

每个项目使用 grid-area 指定它所属的区域名称。

.grid {
  grid-template-areas:
    "header header header"
    "sidebar main aside"
    "footer footer footer";
}

.header { grid-area: header; }
