基础概念

坐标系统

Transform 变换

translate() 平移 translateX() X轴平移 translateY() Y轴平移 rotate() 旋转 scale() 缩放 skew() 倾斜

translateY() - Y轴平移

只在垂直方向（Y轴）上移动元素

语法说明

transform: translateY(y);
  • y - Y 轴方向的偏移量
  • 正值：向下移动（注意：CSS 的 Y 轴正方向是向下的）
  • 负值：向上移动
  • 等价于 translate(0, y)

示例 1：向下移动

translateY(100px) 将元素向下移动 100 像素

X → Y ↓
向下
.box {
  transform: translateY(100px);
  /* 向下移动 100px */
}

示例 2：向上移动

使用负值向上移动

X → Y ↓
向上
.box {
  transform: translateY(-100px);
  /* 向上移动 100px */
}

常见应用场景

  • 下拉菜单展开效果
  • Toast 通知弹出
  • 页面滚动视差效果
  • 元素垂直居中：translateY(-50%) 配合 top: 50%
  • 悬停上浮效果：:hover { transform: translateY(-5px); }

记住：在 CSS 中，Y 轴的正方向是向下的，这与数学坐标系相反！
