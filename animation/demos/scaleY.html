scaleY() - Y轴缩放

只在垂直方向缩放元素

语法说明

transform: scaleY(y);
  • y - Y 轴方向的缩放比例
  • 1：保持原始高度
  • > 1：垂直拉伸
  • < 1：垂直压缩
  • -1：垂直翻转
  • 等价于 scale(1, y)

示例 1：垂直拉伸

scaleY(2) 将元素高度拉伸到 2 倍

2x
.box {
  transform: scaleY(2);
  /* 高度变为 2 倍，宽度不变 */
}

示例 2：垂直压缩

scaleY(0.5) 将元素高度压缩到一半

0.5x
.box {
  transform: scaleY(0.5);
  /* 高度缩小到 50% */
}

示例 3：垂直翻转

scaleY(-1) 垂直镜像翻转元素

-1x
.box {
  transform: scaleY(-1);
  /* 垂直翻转 */
}

提示：scaleY 常用于垂直方向的展开/收起动画
