translateX() - X轴平移

只在水平方向（X轴）上移动元素

语法说明

transform: translateX(x);
  • x - X 轴方向的偏移量
  • 正值：向右移动
  • 负值：向左移动
  • 等价于 translate(x, 0)

示例 1：向右移动

translateX(150px) 将元素向右移动 150 像素

X → Y ↓
向右
.box {
  transform: translateX(150px);
  /* 向右移动 150px */
}

示例 2：向左移动

使用负值向左移动

X → Y ↓
向左
.box {
  transform: translateX(-150px);
  /* 向左移动 150px */
}

常见应用场景

  • 侧边栏滑入滑出效果
  • 轮播图切换动画
  • 水平滚动效果
  • 元素水平居中：translateX(-50%) 配合 left: 50%

性能提示：使用 transform 比修改 left/right 属性性能更好，因为它不会触发页面重排（reflow）
